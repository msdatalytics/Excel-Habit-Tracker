# 📊 Excel Habit Tracker

A simple and clean Excel-based Habit Tracker built from scratch to track daily habits and visualize progress.
This project was created to improve my Excel skills while building something useful for my daily routine.

---

## ✨ Features
- Daily habit tracking with checkboxes  
- Auto-calculated completion percentages  
- Dynamic sorting of top habits  
- Progress bars and charts  
- Simple dashboard view  

---

## 🛠 Built Using
- Microsoft Excel  
- Formulas: COUNTIF, SORT, VLOOKUP, IFERROR  
- Conditional formatting & charts  

---

## 🎯 Purpose
Instead of using a ready-made template, I built this tracker myself to:
- Practice Excel automation  
- Improve dashboard-building skills  
- Create a personal productivity tool
